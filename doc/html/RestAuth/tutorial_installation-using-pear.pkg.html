Installation using PEAR

How-to install using PEAR

Mathias Ertl
mati@restauth.net

You can also install php-restauth using PEAR. We provide our own PEAR channel containing stable releases of php-restauth. First, you must discover our channel:

pear channel-discover pear.php.restauth.net

and then install the package simply by doing:

pear install RestAuth/Authentication_RestAuth

Don't forget to install pecl_http.

Upgrading

To upgrade all packages, simply run:

pear upgrade

If you only want to upgrade this package, do:

pear upgrade RestAuth/Authentication_RestAuth
