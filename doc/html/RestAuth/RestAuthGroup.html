static method createTest (line 119)

Test if creating a group with the current parameters would succeed or not. This method always returns true if the group could be created, otherwise it should throw the exact same exceptions as if you would actually create the group.

Note that doing this request never guarantees that an actual request works in the future, it can only assure that it would succeed right now.

  • return: Always returns true if the succeed or not.
