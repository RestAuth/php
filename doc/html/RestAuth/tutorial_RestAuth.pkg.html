Welcome to the homepage of php-restauth, a client library for RestAuth. The library allows systems to share user authentication, authorization and preferences on a RestAuth server. The php-restauth library hides all protocol and network details from the programmer, enabling fast, efficient and easy-to-maintain integration into existing services.

php-restauth is written in pure PHP. php-restauth is free software and licenced under the GNU General Public Licence, version 3, so please feel free to contribute.
