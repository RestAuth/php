Contribute

How-to contribute

Mathias Ertl
mati@restauth.net

php-restauth is Free Software and as such invites everyone to contribute in any way possible.

  • If you find bugs or want to request new features, you can simply file a new issue in our issue tracker.

  • You can also join our XMPP/Jabber multi user chat at rest@conference.jabber.at. If you don't have an account, you can join it without an account using speeqe.

  • Our codebase is hosted at git.fsinf.at/restauth/php. You can fork our codebase and code your own improvements and do a merge-request to get your changes into the official codebase.

    Note that git.fsinf.at already uses a RestAuth installation. Account registration is not possible on git.fsinf.at itself, please register here.
