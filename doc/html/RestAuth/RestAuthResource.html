Class RestAuthResource

Description
Description | Descendents | Methods (details)

Superclass for RestAuthUser and RestAuthGroup objects.

Exists to wrap http requests with the prefix of the given resource.

		deleteRequest (line 641)
	
 
	
	

Perform a DELETE request on the connection that was passed via the  constructor.


This method prefixes the URL parameter with the the resources class  prefix ('/users/' or '/groups/') and passes all parameters (otherwise  unmodified) to RestAuthConnection::delete().

	
    
				
  • return: The response to the request.
    • 
				
  • link: HttpMessage
    • 
				
  • throws: RestAuthUnauthorized When service authentication      failed.
    • 
				
  • throws: RestAuthNotAcceptable When the server cannot generate     a response in the content type used by this connection.
    • 
				
  • throws: RestAuthInternalServerError When the RestAuth service     suffers from an internal error.
    • 
				
  • throws: RestAuthRuntimeException When some HTTP related error     occurs.
